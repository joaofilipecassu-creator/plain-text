# Projeto - Exercícios JavaScript

Este projeto foi desenvolvido utilizando **HTML, CSS e JavaScript** com o objetivo de praticar conceitos básicos e intermediários de programação web, manipulação de elementos e interação com o usuário.

## 📚 Objetivo

O trabalho tem como foco o aprendizado de:

* Estruturação de páginas HTML
* Estilização com CSS
* Lógica de programação em JavaScript
* Manipulação de datas e horários
* Funções e interação dinâmica na página

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## ⚙️ Funcionalidades

* Exibição dinâmica de saudação
* Manipulação de data e hora
* Organização de dias da semana
* Interações simples utilizando JavaScript

## 📁 Estrutura do Projeto

```bash
📦 projeto
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┗ 📄 script.js
```

## 🚀 Como Executar

1. Faça o download ou clone este repositório:

```bash
git clone https://github.com/joaofilipecassu-creator/seu-repositorio.git
```

2. Abra o arquivo `index.html` no navegador.
